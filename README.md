# Findex Web app
Vue 3 [Metronic](https://keenthemes.com/metronic/) `v8.1.4` app

## Prerequisties
 - Node `17` (Node version handling [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

## How to run
1. Install modules
```
npm install
```
1. Run in dev mode (`http://localhost:8081`)
```
npm run dev
```
